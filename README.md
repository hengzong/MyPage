# To Get Start
#### Step 1: Start the Backend Server

```sh
(MyPage)$ cd backend
(backend)$ npm install
(backend)$ node server.js
```

#### Step 2: Start the Frontend Server
```sh
(MyPage)$ cd simple_homepage
(simple_homepage)$ npm install
(simple_homepage)$ npm start
```

#### Step 3: Go to http://localhost:3000
